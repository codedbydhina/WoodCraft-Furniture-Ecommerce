# 🌳 WoodCraft – Premium Wooden Furniture E-Commerce Web Application

## 📌 Overview

WoodCraft is a full-stack e-commerce web application designed for purchasing premium wooden furniture online. The application provides a seamless shopping experience for customers while offering a comprehensive admin panel for managing products, inventory, and orders.

---

## 🚀 Features

### Customer

* User Registration & Login
* Browse Products
* Product Search
* Product Details
* Shopping Cart
* Place Orders
* View Order History
* Responsive User Interface

### Admin

* Secure Admin Login
* Dashboard
* Add, Update & Delete Products
* Manage Product Inventory
* View & Manage Customer Orders

## ⚙️ Installation

1. Clone the repository.

```bash
git clone https://github.com/your-username/woodcraft.git
```

2. Import the project into Eclipse.

3. Create a MySQL database.

4. Import `woodcraft.sql`.

5. Update database credentials in the JDBC configuration.

6. Deploy the project on Apache Tomcat.

7. Open the application in your browser.
